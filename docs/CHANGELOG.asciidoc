== Version 4.5.0 _(2015-06-15)_

.Additions and changes:
* Add Bootswatch Darkly theme to standard distribution
* Add Off-canvas menu style for the Table Of Contents (new attribute `tocmmenu`)
+
CAUTION: Standalone document is not supported (used `linkcss` combined with `tocmmenu`)

.Bug fixes:
* https://github.com/llaville/asciidoc-bootstrap-backend/issues/16[GH-16] : Fixed subtitle syntax issue (`= title @subtitle@`).

== Version 4.4.0 _(2015-04-04)_

.Additions and changes:
* Ability to add an icon (iconfont) in the header of a panel component
* Responsive-option in image inline macro was removed (see https://github.com/llaville/asciidoc-bootstrap-backend/issues/15[GH-15])
* Add `align` option in image block macro to center image in a div
* Upgrade Bootstrap to version 3.3.4
* Upgrade Bootswatch themes (Flatly, Readable, Cerulean, United, Sandstone) to version 3.3.4

.Bug fixes:
* Fixed theme switcher usage.
* Fixed render of Panel without title.
See commit https://github.com/llaville/asciidoc-bootstrap-backend/commit/7a8f39e0994926821aa0c225c77785465ade6a8a[7a8f39e]

== Version 3.2.0 _(2013-12-20)_

.Additions and changes:
* Upgrade Bootstrap and Bootswatch Themes to latest version 3.0.3
* New `favicon` attribute to specify a favorite icon, if any.
* AsciiDoc callouts feature used badges rather than images.
* `totop` attribute does not have values anymore. Sets for UI behavior and unset for text behavior.

.Bug fixes:
* none

== Version 3.1.0 _(2013-12-03)_

.Additions and changes:
* CSS signature, AsciiDoc feature, is now available with the bootstrap backend.
* Open Blocks allows to write blog, releases or features pages.
* Ability to use all Pygments styles (globally or locally).
* Add an inline macro to represent an icon.
* Add Cerulean theme from Bootswatch template provider.
* Add backend build signature in text footer.
* Upgrade Bootstrap and Bootswatch Themes to latest version 3.0.2
* New `rssref` attribute to specify a RSS feed in output document.
* Admonition Block support text, images and icons font.
* Optional additional CSS stylesheet, AsciiDoc feature, is now available with the bootstrap backend.

.Bug fixes:
* none
